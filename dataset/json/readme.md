<h2 id="overview">About_Data</h2>

<h3 id="philosophy">Format</h3>

	1. 格式：JSON
	2. 欄位架構:
		> 1. timestamp: 發表該推文的時間，年-月-日T時:分:秒.毫秒時區
		> 2. message: 文章內容
		> 3. user: 作者
		> 4. client: 使用的客戶端(for Mac/iPhone/iPad/Android等等)
		> 5. retweet: 是否被轉推(true/false)
		> 6. source: 文章的網址
		> 7. hashtags: 文章是否存在標籤
		> 8. symbols: 符號(表情/物件/地點符號等等)
		> 9. user_mention: 用@提到的內容
		>10. version: 版本
